# RFO UI Emulator ‚Äî Project Instructions

## Overview
Interactive web-based emulator for RF Online interface concepts. No backend, no build step ‚Äî just HTML/CSS/JS served from a static host (GitHub Pages).

## Architecture
```
index.html                ‚Üê Entry point, viewport, control panel
css/core.css              ‚Üê Global styles, CSS variables, panel theme
js/app.js                 ‚Üê Bootstrap: loads manifest, inits managers
js/core/
  settings.js             ‚Üê Global state (scale, bounds, mode), localStorage
  window-manager.js       ‚Üê Window registry, open/close/focus, z-index stack
  drag-engine.js          ‚Üê Pointer-event drag with screen bounds
  resize-engine.js        ‚Üê Resize handles, min/max constraints
  context-menu.js         ‚Üê Custom right-click menu
  layout-manager.js       ‚Üê Preset save/load, URL sharing, auto-save
  comment-manager.js      ‚Üê Pin annotations on windows
  export-manager.js       ‚Üê html2canvas PNG export, batch ZIP
windows/
  manifest.json           ‚Üê Registry of all windows
  _template/              ‚Üê Boilerplate for new windows (see WINDOW.md)
  {window-id}/            ‚Üê Each window: config.js, template.html, style.css
presets/                  ‚Üê Built-in preset files
assets/backgrounds/       ‚Üê Background images/videos
```

## How to Create a New Window
1. Copy `windows/_template/` ‚Üí `windows/{your-window-id}/`
2. Edit the 3 files: `config.js`, `template.html`, `style.css`
3. Add an entry to `windows/manifest.json`
4. See `windows/_template/WINDOW.md` for full documentation

## CSS Variables Reference (defined in core.css :root)
- `--ui-scale` ‚Äî Current scale factor (1.0 = 100%)
- `--viewport-w`, `--viewport-h` ‚Äî Base viewport dimensions (1920, 1080)
- `--z-background`, `--z-windows`, `--z-overlays`, `--z-context-menu`, `--z-panel`
- `--panel-bg`, `--panel-border`, `--panel-accent`, `--panel-text`

## Core APIs Available to Windows

### WindowManager (`js/core/window-manager.js`)
```js
import { windowManager } from '../../js/core/window-manager.js';
windowManager.open(id)      // Open window
windowManager.close(id)     // Close window
windowManager.toggle(id)    // Toggle open/close
windowManager.focus(id)     // Bring to front
```

### Settings (`js/core/settings.js`)
```js
import { settings } from '../../js/core/settings.js';
settings.get('scale')       // Current scale factor
settings.get('mode')        // 'design' | 'export' | 'comment'
settings.on('mode', fn)     // Subscribe to changes
```

## Key Decisions
- **No Shadow DOM**: CSS scoped via `[data-window-id]` attribute selectors for html2canvas compatibility
- **No framework**: Vanilla JS + ES Modules for maximum simplicity
- **Base resolution**: 1920√ó1080, scaled via CSS transform
- **Export format**: PNG with transparency via html2canvas
- **Comments**: Pin annotations stored in presets, shared via JSON/URL
